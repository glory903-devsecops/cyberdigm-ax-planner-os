# CMTX AX Planner OS - 제품 요구사항 문서 (PRD)

## 1. 프로젝트 개요
반도체 R&D 및 산업 동향을 자동 수집하고 분석하는 엔터프라이즈 전략 플랫폼.

## 2. 핵심 기능 요구사항

### 2.1. 자동화 센터 (Automation Center)
- **수집 타겟 등록**: 사용자는 사이트명, URL, 키워드, 수집 주기 등을 설정하여 새로운 크롤링 타겟을 등록할 수 있어야 함.
- **조건부 저장**: 
    - 관리자 로그인 상태: Supabase DB에 영구 저장.
    - 게스트 상태: 로컬 시뮬레이션(새로고침 시 초기화) 및 안내 메시지 출력.
- **UI**: FAB(+) 버튼을 통한 모달 기반 등록 폼 제공.

### 2.2. GR Hub 및 대시보드
- **인터랙티브 리스트**: 대시보드의 지원사업/정책 리스트 항목은 독립적으로 클릭 가능해야 함.
- **상세 패널**: 항목 클릭 시 우측에서 슬라이딩 상세 패널이 열리며 원문 링크 및 AI 요약 정보를 제공함.
- **실시간 피드**: 24시간 자동 수집 상태를 시각적으로 표시함.

### 2.3. 인증 시스템
- **관리자 로그인**: 사이드바 하단의 로그인 버튼을 통해 로컬 액세스 키로 인증 수행.
- **권한 제어**: 로그인 여부에 따라 '관리자 모드'와 '게스트 모드'를 시각적으로 구분하여 표시.

## 3. UI/UX 원칙
- **프리미엄 에스테틱**: 현대적이고 고급스러운 다크/블루 테마 적용.
- **부드러운 인터랙션**: Framer Motion을 활용한 애니메이션 효과.
- **반응형 디자인**: 태블릿 크기(768px)까지 호버링 사이드바 유지.

## 4. 기술 스택
- Frontend: Next.js 15+, TailwindCSS, Framer Motion
- Backend/DB: Supabase
- Crawler: Python based (bizinfo, spri 등)
